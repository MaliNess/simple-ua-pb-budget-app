Import XLSX

The first two rows are ignored. Data is read from row 3 in this order:

  1. Date
  2. Category
  3. Card
  4. Description
  5. Plain transaction sum
  6. Plain currency
  7. Initial transaction sum
  8. Initial currency
  9. Remaining sum
  10. Remaining currency

Imported tickets are added to Unassigned. A leading minus sign in the plain transaction sum is removed automatically.

The built-in importer reads the first worksheet of standard, non-encrypted .xlsx files. Legacy .xls files are not supported.
